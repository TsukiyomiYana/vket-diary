# Vket 2026 Summer 參展日記

這是我第一次參加 Vket（Virtual Market／バーチャルマーケット）。

為了避免未來的自己又忘記當初究竟是怎麼操作的、\
我打算好好地、認真地、盡可能完整地、把從準備到實際參展的流程記錄下來。

裡面會包含~~大量的廢話~~、製作與參展的過程、\
途中遇到的問題與解法、以及全部完成後的心得。

希望這份紀錄能幫助到同樣想參加 Vket 的創作者。

> ※ 這不是正式教學、只是單純的個人分享與參展紀錄。\
> ※ 由於一開始沒有寫紀錄的計畫、圖片多半是事後翻找或補拍、內容可能不夠完整，還請見諒。
