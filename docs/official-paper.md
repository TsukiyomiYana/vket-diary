# 官方必讀文件

製作之前記得要好好閱讀官方文件說明、才不會白做工。

1. [出展規約](https://account.vket.com/terms?locale=ja#event)\
參展的法律條款與規定、報名前一定要讀過、確認自己同意所有條款。
2. [出展指南（出展ガイドライン）](https://vket.com/docs/2026summer/exhibition_guideline)\
參展的總覽說明、包含展出形式（Space、Item、Avatar）的介紹、\
各個世界觀的設定、入稿期間與整個活動的規劃日程等。
3. [交稿指南（入稿ガイド）](https://vket.com/docs/2026summer/submission_guide_line)\
各個展出形式的入稿步驟教學、針對你報名的產出方式閱讀即可。\
也可以搭配官方教學影片：[Vket 2026 Summer 入稿ガイド - アイテム出展](https://youtu.be/ghvxqGDO8j8?si=DCuy1AxRAPSwl_yW)
4. [交稿規則（入稿ルール）](https://vket.com/docs/2026summer/submission_rules)\
確認入稿的技術規格限制、例如檔案大小、面數限制、Shader 使用規範、禁止事項等。

> ※官方文件內容可能會隨時更新、報名前與入稿前請務必再次確認最新版本。

## 給新手的製作輔助資料

* [從零開始的 Vket 入稿基礎（0から始めるVket入稿 基礎知識編）](https://vket.com/docs/zero_vket_base)\
針對不熟悉 Unity 的出展者、解釋 VketTools、Asset 匯入、Hierarchy 等基本概念。
* [VketTools 安裝流程（VketTools インストール手順）](https://vket.com/docs/vkettools_how_to_install)\
VketTools 是入稿必備的工具、\
官方文件有非常詳細的教你如何在 VRChat Creator Companion (VCC) 中安裝它。
* [技術指南（テクニカルガイド）](https://vket.com/docs/submission_tips)\
大部分製作過程中遇到效能或視覺問題、都能在這裡找到答案。
* [官方素材包（Vket 公式アセット）](https://vket.com/docs/official_distribution_package)\
Vket 官方提供的 Prefab 素材、包含 Avatar Pedestal、Pickup 物件、影片播放器等常用功能。\
使用官方 Prefab 可以省下很多時間、而且部分 Unity 功能只有透過官方 Prefab 才能使用。
