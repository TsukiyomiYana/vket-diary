# Vket 報名與抽選

這邊僅簡單記錄一下我報名 Vket 前做了哪些事情。

首先、需要先到 [Vket](https://vket.com/) 官網申請一個 Vket Account。\
其次、因為 Vket 本身是在 VRChat 裡舉辦的活動、後面展示確認、測試或實際逛展時都會用到。\
所以也建議先申請好 [VRChat](https://hello.vrchat.com/) 帳號。

## 建立サークル (Create a new circle)

帳號建立好之後、\
在側邊選單裡點選「プロフィール (User Profile)」、編輯自己的個人資料。

<StepGallery arrows={false} maxWidth="480px" steps={[
  { img: '/vket-diary/img/web-003.webp', label: 'User Profile' }
]} />

接著到「サークル管理 (Exhibitor Circles)」裡選擇「新規登録 (Create a new circle)」、\
把サークル資料填好並送出之後、就成功建立自己的 Circle 啦。

<StepGallery steps={[
{ img: '/vket-diary/img/web-004.webp', label: 'Create a New Circle' },
{ img: '/vket-diary/img/web-005.webp', label: 'Enter Circle Information' },
]} />

如果是和夥伴一起參展、可以在 Circle 頁面裡找到「メンバー管理 (Edit Members)」、\
接著選擇「メンバー追加 (Add Member)」、輸入夥伴的 Vket ID 並送出邀請。\
邀請送出後、你的夥伴需要到自己的「サークル管理」裡接受邀請、\
完成之後、你的夥伴就正式加入你的 Circle 囉。

**報名參展前一定要先建立 Circle、所以這一步是必須完成的。**

## 申請參展（Exhibition Application）

Circle 建立完成之後、就可以到「出展申し込み (Exhibition Application)」進行參展申請。\
申請頁面裡會有不同的展區與出展形式可以選擇、選好想報名的展區與出展形式之後、就可以送出申請。

{ img: '/vket-diary/img/web-006.webp', label: 'Exhibition Application' },

如果不確定要選哪個展區、~~但又希望提高中獎的機率~~、\
也可以參考[Vket官方](https://x.com/Virtual_Market_)在 X 上於報名期間公開的「出展申し込み中間発表」。\
大概看一下各個展區目前的申請情況、再決定要不要調整自己的申請目標。

> ~~※ 每次活動的狀況可能都不一樣、還是要以當期官方公告為準。~~\
~~※ 有都市傳說提到、抽選會以第1希望 (1st Choice) 為主、所以...第1希望要好好選。~~

選好想報名的展區與出展形式之後、送出申請、接著就是等待抽選結果。\
一共會有2次的抽選機會。
