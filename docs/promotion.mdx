# 社群宣傳紀錄

眾所皆知 (?)、我是一個把 BOOTH 商店放置 PLAY 不管的人（ry\
換句話說、推廣作品這件事我不太拿手、~~所以早就做好會涼透的心理準備了。~~

不過、為了爆★誕的臭豆腐海兔、~~硬著頭皮~~我還是好好規劃一下、\
試著讓更多人知道、~~臭豆腐超好吃的~~、\
咳...我是說、我的臭豆腐海兔到底有多可愛、又能拿來做多少有趣的事情。

為了這次上架、我委託月兔 ([@Totoyue_VRC](https://x.com/Totoyue_VRC)) 拍攝了幾張超美的宣傳圖、\
也在實測過程中意外錄到了兩支趣味影片。

除此之外、我自己也拍了兩支短片、\
一支是操作展示、另一支則是比較輕鬆有趣 (?) 的內容。

因此、我打算將不同的內容拆開、\
每隔幾天發布一支、分別介紹基本操作、坐姿、Avatar 版本、以及各種意外誕生的玩法、\
一路持續到 7 月 11 日正式開賣。

年紀大了，實在經營不了太多平台（ry\
所以宣傳還是集中在 X 上。
