# 臭豆腐海兔的誕生

這次抽選到的世界是「[幻想コロニー コクーン水族美術館 Redive - パルス](https://vket.com/2026Summer/world/29)」。

最初想著、水族美術館啊...\
或許可以做一個跟世界觀有關的小東西。

想像力匱乏如我、\
單純地在青蛙、海豚、魟魚和海兔之間猶豫了一陣子。

在網路上隨手搜尋資料、結果卻不知不覺蒐集了一大堆海兔的參考圖、\
其他幾個候選反而連一張都沒有留下來。

...呃、感覺答案好像已經很明顯了。XD"

<StepGallery arrows={false} maxWidth="600px" steps={[
  { img: '/vket-diary/img/web-014.webp', label: 'Umiushi Reference' }

於是、最後就決定以海兔作為主題。\
方向決定之後、我畫了幾個不同造型的海兔草稿。

<StepGallery arrows={false} maxWidth="600px" steps={[
  { img: '/vket-diary/img/web-015.webp', label: 'Umiushi Sketch' }
]} />

畫著畫著、突然覺得牠們看起來有點像小寶寶、又有點像某些食物。\
~~絕對不是因為我當時又睏又餓之類的...~~

於是、我畫了寶寶海兔、蜂蜜蛋糕海兔、又畫了枕頭海兔。

<StepGallery arrows={false} maxWidth="600px" steps={[
  { img: '/vket-diary/img/web-016.webp', label: 'Baby Umiushi / Food Umiushi' }
]} />

接著、又想到幾年前還能吃麻辣臭豆腐的時候、\
那個黑色的臭豆腐真的很好吃...

~~這個人在畫圖的時候、腦子通常都不太正常。~~ `( °ヮ° ) ?`

給另一半和身邊幾位朋友看過之後、\
大部分的人好像都比較喜歡寶寶海兔。

原本想著、既然如此、那就做寶寶海兔吧。

結果、寶寶海兔的模型都已經做完了、\
我心裡卻還是更偏心黑色的臭豆腐海兔。

~~臭豆腐真的很好吃~~。

所以最後、還是臨時鬼轉、
決定順著自己的喜好去做臭豆腐海兔。\

於是、臭豆腐海兔就這樣誕生了。
