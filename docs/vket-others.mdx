# Vket 後台備忘錄

## 下見ワールド（Preview World）

在 **入稿 (Submission) → 入稿 / 修正 (Submission / Edit) → 入稿方法 (Submission Method)** 中、\
可以找到 **下見ワールドへ (Enter the Preview World)** 的按鈕。

按下後會先顯示 **入場規約 (Entry Terms for the Preview World)** 。\
閱讀並同意後、即可進入下見ワールド、確認自己的商品在實際展場中的顯示與操作狀況。

入場規約務必仔細確認。\
若違反規約、除了可能被**取消參展資格**之外、根據造成的損害程度、甚至可能被追究**法律責任**。
