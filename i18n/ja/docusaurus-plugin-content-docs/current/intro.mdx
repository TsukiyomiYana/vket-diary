# Vket 2026 Summer 出展日記

これは、初めての Vket（Virtual Market / バーチャルマーケット）出展記録です。

未来の自分が、当時いったいどうやって進めたのかをまた忘れてしまわないように、\
準備から実際の出展までの流れを、きちんと、真面目に、できる限り詳しく記録しておくことにしました。

中身には~~大量の無駄話~~や、出展・制作の過程、途中でぶつかった問題とその解決方法などが含まれています。

この記録が、同じように Vket に出展したいと思っているクリエイターの役に立ち、背中を押すことができれば嬉しいです。

> ※ これは正式なチュートリアルではなく、あくまで個人的な体験の共有と出展記録です。\
> ※ 最初は記録を書く予定がなかったため、画像の多くは後から探したり、改めて撮影したりしたものです。内容に不完全な部分があるかもしれませんが、ご了承ください。\
> ※ 日本語版と英語版は、Claude に翻訳を手伝ってもらいました。今後、時間ができたときに日英版を詳しく見直す予定です。


## 出展日記について

この記録には、以下の内容が含まれています：

### 出展準備

* **Vket 出展のきっかけ** — Vket を知ってから、出展を決めるまでの過程
* **Vket 応募と抽選** — 応募の流れ、サークルの作成と抽選結果
* **スケジュール表** — 制作スケジュールと、各段階での実際の進捗
* **Vket 管理メモ** — 下見ワールドとカタログ管理の操作記録

### 制作・つまずきメモ

* **スティウミ誕生** — デザイン、3D モデリングからテクスチャ制作までの WIP 記録
* **入稿つまずきガイド** — 入稿時にぶつかった問題、具体的な解決方法と注意点
* **商品ファイル血涙史** — 販売用 Unity Package を整理・制作したときのつまずき記録
* **ウミウシの展示スペース** — 展示用ワールドの構築と機能制作の記録

### 販売・宣伝と資料メモ

* **BOOTH 商品ページ** — 商品ページ、画像、動画の整理記録
* **SNS 宣伝記録** — X での宣伝計画と投稿記録
* **使用したツールと素材** — 制作過程で使用したツール、素材、参考資料
* **公式必読ドキュメント** — Vket の出展・制作・入稿時に参考にした公式チュートリアルとガイド
* **スペシャルサンクス** — 制作過程で助けてくれたすべての人への感謝

### DLC：展示スペースのちょっとした隠し要素
