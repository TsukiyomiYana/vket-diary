# Vket 応募と抽選

ここでは、Vket に出展申し込みをする前にやったことを、簡単に記録しておきます。

まず、[Vket](https://vket.com/) 公式サイトで Vket Account を作成する必要があります。\
次に、Vket は VRChat 内で開催されるイベントなので、後ほど展示の確認やテスト、実際に会場を見て回るときにも VRChat を使います。\
そのため、[VRChat](https://hello.vrchat.com/) のアカウントも先に作っておくことをおすすめします。

## サークルを作成する (Create a new circle)

アカウントを作成したら、\
サイドメニューで「プロフィール (User Profile)」を選び、自分のプロフィールを編集します。

<StepGallery arrows={false} maxWidth="480px" steps={[
  { img: '/vket-diary/img/web-003.webp', label: 'User Profile' }
]} />

次に「サークル管理 (Exhibitor Circles)」で「新規登録 (Create a new circle)」を選び、\
サークル情報を入力して送信すれば、自分のサークルが作成できます。

<StepGallery steps={[
{ img: '/vket-diary/img/web-004.webp', label: 'Create a New Circle' },
{ img: '/vket-diary/img/web-005.webp', label: 'Enter Circle Information' },
]} />

仲間と一緒に出展する場合は、サークルページにある「メンバー管理 (Edit Members)」から、\
「メンバー追加 (Add Member)」を選び、相手の Vket ID を入力して招待を送ります。\
招待を送ったあと、相手にも自分の「サークル管理」から招待を承認してもらう必要があります。\
承認が完了すると、その相手が正式にサークルのメンバーになります。

**出展申し込みの前に必ずサークルを作成する必要があるので、このステップは必須です。**

## 出展申し込み（Exhibition Application）

サークルを作成したら、「出展申し込み (Exhibition Application)」から出展を申請できます。\
申請ページでは、複数の出展エリアと出展形式から、希望するものを選んで申し込みます。

<StepGallery steps={[
{ img: '/vket-diary/img/web-006.webp', label: 'Exhibition Application' },
{ img: '/vket-diary/img/web-007.webp', label: 'Application Submitted' },

どのエリアを選べばいいか分からない、~~でも少しでも当選確率を上げたい~~という場合は、\
[Vket 公式](https://x.com/Virtual_Market_) が募集期間中に X で公開している「出展申し込み中間発表」も参考にできます。\
各エリアの現在の応募状況をざっくり確認してから、自分の応募先を調整するかどうか決めるのもいいと思います。

<StepGallery arrows={false} maxWidth="480px" steps={[
  { img: '/vket-diary/img/web-008.webp', label: '出展申し込み中間発表' }

> ~~※ イベントごとに状況は異なる可能性があるため、必ずその回の公式発表を確認してください。~~\
~~※ 抽選は第1希望を中心に行われるという都市伝説があるので...第1希望は慎重に選びましょう（迷信）。~~

出展エリアと形式を選んで申請を送ったら、あとは抽選結果を待つだけです。\
抽選のチャンスは全部で2回あります。
