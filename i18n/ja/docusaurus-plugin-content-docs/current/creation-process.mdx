# スティウミ誕生

抽選で決まったワールドは「[幻想コロニー コクーン水族美術館 Redive - パルス](https://vket.com/2026Summer/world/29)」でした。

最初は「水族美術館かぁ...\
世界観に関係するちょっとしたものを作るのもいいかもしれない」と思いました。

想像力の乏しい私は、\
カエル、イルカ、エイ、ウミウシのどれにするか、しばらく迷っていました。

ネットで何となく資料を探していたら、気づけばウミウシの参考画像ばかり大量に集まっていて、\
他の候補は一枚も保存されていませんでした。

...えっと、答えはもう出ていたようです。XD"

<StepGallery arrows={false} maxWidth="600px" steps={[
  { img: '/vket-diary/img/web-014.webp', label: 'Umiushi Reference' }
]} />

というわけで、テーマはウミウシに決定。\
方向性が決まったので、デザインの違うウミウシのラフをいくつか描きました。

<StepGallery arrows={false} maxWidth="600px" steps={[
  { img: '/vket-diary/img/web-015.webp', label: 'Umiushi Sketch' }
]} />

描いているうちに、なんだか赤ちゃんのようにも、何かの食べ物のようにも見えてきました。\
~~当時、眠くてお腹が空いていたからでは決してありません...~~

そこで、ベビーウミウシ、カステラウミウシ、まくらウミウシを描きました。

<StepGallery arrows={false} maxWidth="600px" steps={[
  { img: '/vket-diary/img/web-016.webp', label: 'Baby Umiushi / Food Umiushi' }
]} />

次に、数年前、まだ麻辣臭豆腐を食べられた頃のことを思い出しました。\
あの黒い臭豆腐、本当においしかったな...

~~この人、絵を描いているときは、だいたい頭がまともじゃない。~~ `( °ヮ° ) ?`

パートナーと身近な友人何人かに見せたところ、\
ほとんどの人がベビーウミウシのほうを気に入ったようでした。

それなら、ベビーウミウシを作ろうと思いました。

ところが、ベビーウミウシのモデルが完成しても、\
心の中では、まだ黒い臭豆腐ウミウシのほうに気持ちが傾いていました。

~~臭豆腐って、本当においしいな~~。

というわけで最後は、急きょ方向転換し、\
自分の好みに従ってスティウミ（スティンキー豆腐ウミウシ）を作ることにしました。

こうして、スティウミが誕生しました。
