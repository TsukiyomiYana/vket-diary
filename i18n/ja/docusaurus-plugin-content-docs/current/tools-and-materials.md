# 使用ツールと素材

スティウミ本体の制作から Vket への入稿、VRChat の展示スペース制作まで、\
以下のツールと素材を使用しました。

### 制作ツール

* [Blender 5.0](https://www.blender.org/)
* [CLIP STUDIO PAINT (5.0)](https://www.clipstudio.net/tc/)
* [Adobe Photoshop](https://www.adobe.com/tw/products/photoshop.html)

### VRChat / Unity ツール

* [VRChat Creator Companion（VCC）](https://vcc.docs.vrchat.com/) / Unity 2022.3.22f1
* [lilToon (Vket 2.3.2 / Booth 2.3.4)](https://lilxyzw.github.io/lilToon/)
* [Modular Avatar](https://modular-avatar.nadena.dev/) (Avatar 版の MA Toggle 設定)
* [AvaPo！｜Unityでポーズ作成＆撮影ツール](https://booth.pm/ja/items/7249970) (Unity 内でポーズを作成・撮影するためのツール)
* [LuraSwitch2](https://booth.pm/ja/items/1969082) (VRChat 展示スペース内のギミック設定)
