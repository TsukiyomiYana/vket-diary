もともとは、Vket 入稿バージョンが完成したら、\
あとは商品ファイルを少し整理すれば、だいたい終わるはずでした。

ところが、BOOTH 商品を整理している途中で、\
予定になかった Avatar 版まで追加で作ることにしました。

さらに World 版の Sit Prefab を整理していたら、\
ふと「いろいろな座りポーズを切り替えられたら、もっと面白そう！」と思ってしまいました。

そこからポーズ切り替えの方法を調べ、スクリプトを書き、メニューを作り、8種類の座りポーズを自作し、\
さらにメニューが VRChat のカメラに写り込まないようにするシェーダーまで書き...

最後には、実際に試せる展示スペースまで作ってしまいました...Orz
