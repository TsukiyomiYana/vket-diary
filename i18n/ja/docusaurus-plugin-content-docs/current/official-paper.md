# 公式必読ドキュメント

制作を始める前に、公式ドキュメントをしっかり読んでおきましょう。せっかくの作業が無駄にならないように。

1. [出展規約](https://account.vket.com/terms?locale=ja#event)\
出展に関する法的条項とルール。応募前に必ず読んで、すべての条項に同意できるか確認しましょう。
2. [出展ガイドライン](https://vket.com/docs/2026summer/exhibition_guideline)\
出展の全体像。出展形式（Space、Item、Avatar）の紹介、\
各ワールドの世界観、入稿期間、イベント全体のスケジュールなどがまとめられています。
3. [入稿ガイド](https://vket.com/docs/2026summer/submission_guide_line)\
各出展形式の入稿手順がまとめられています。自分が申し込んだ出展形式の部分を読めば OK です。\
公式チュートリアル動画もあわせて確認できます：[Vket 2026 Summer 入稿ガイド - アイテム出展](https://youtu.be/ghvxqGDO8j8?si=DCuy1AxRAPSwl_yW)
4. [入稿ルール](https://vket.com/docs/2026summer/submission_rules)\
ファイルサイズやポリゴン数の上限、Shader の使用ルール、禁止事項など、入稿時の技術的な制限を確認します。

> ※ 公式ドキュメントの内容は随時更新される可能性があります。応募前と入稿前には、必ず最新版を再確認してください。

## 初心者向けの制作サポート資料

* [0から始めるVket入稿 基礎知識編](https://vket.com/docs/zero_vket_base)\
Unity に不慣れな出展者向けに、VketTools、アセットのインポート、Hierarchy などの基本概念を解説しています。
* [VketTools インストール手順](https://vket.com/docs/vkettools_how_to_install)\
VketTools は入稿に必須のツールです。\
VRChat Creator Companion (VCC) でのインストール方法が、公式ドキュメントで非常に丁寧に説明されています。
* [テクニカルガイド](https://vket.com/docs/submission_tips)\
制作中に出てくるパフォーマンスや見た目の問題の多くは、ここで解決方法を見つけられます。
* [Vket 公式アセット](https://vket.com/docs/official_distribution_package)\
Vket 公式が提供している Prefab 素材。Avatar Pedestal、Pickup オブジェクト、動画プレイヤーなど、よく使う機能が含まれています。\
公式 Prefab を使えばかなり時間を短縮できますし、公式 Prefab 経由でしか使えない Unity の機能もあります。
