# Vket 管理メモ

## 下見ワールド（Preview World）

**入稿 (Submission) → 入稿 / 修正 (Submission / Edit) → 入稿方法 (Submission Method)** の中に、\
**下見ワールドへ (Enter the Preview World)** のボタンがあります。

押すと、まず **入場規約 (Entry Terms for the Preview World)** が表示されます。\
内容を読んで同意すれば下見ワールドに入場でき、自分の商品が実際の会場でどのように表示され、動作するかを確認できます。

入場規約は必ずしっかり確認しましょう。\
規約に違反した場合、**出展資格を取り消される**可能性があります。さらに、損害の程度によっては**法的責任**を問われることもあります。
