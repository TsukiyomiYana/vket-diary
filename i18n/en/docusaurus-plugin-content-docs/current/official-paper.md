# Official Must-Reads

Remember to read the official documentation carefully before you start, so none of your work goes to waste.

1. [出展規約 (Exhibition Terms)](https://account.vket.com/terms?locale=ja#event)\
The legal terms and rules for exhibiting. Read them before applying, and make sure you agree to all of them.
2. [出展ガイドライン (Exhibition Guidelines)](https://vket.com/docs/2026summer/exhibition_guideline)\
An overview of exhibiting, including the exhibit formats (Space, Item, Avatar),\
the setting of each world, the submission period, and the full event schedule.
3. [入稿ガイド (Submission Guide)](https://vket.com/docs/2026summer/submission_guide_line)\
Step-by-step submission instructions for each exhibit format. You only need to read the section for the format you applied for.\
You can also refer to the official tutorial video alongside it: [Vket 2026 Summer 入稿ガイド - アイテム出展](https://youtu.be/ghvxqGDO8j8?si=DCuy1AxRAPSwl_yW)
4. [入稿ルール (Submission Rules)](https://vket.com/docs/2026summer/submission_rules)\
The technical requirements and restrictions for submission, such as file-size and polygon-count limits, Shader usage rules, and prohibited practices.

> ※ Official documents may be updated at any time — always re-check the latest version before applying and before submitting.

## Production Resources for Beginners

* [0から始めるVket入稿 基礎知識編 (Vket Submission from Zero: The Basics)](https://vket.com/docs/zero_vket_base)\
For exhibitors who are unfamiliar with Unity, this explains basic concepts such as VketTools, importing assets, and the Hierarchy.
* [VketTools インストール手順 (VketTools Installation Guide)](https://vket.com/docs/vkettools_how_to_install)\
VketTools is a required tool for submission.\
The official documentation explains in detail how to install it through VRChat Creator Companion (VCC).
* [テクニカルガイド (Technical Guide)](https://vket.com/docs/submission_tips)\
You can find answers here for most performance or visual issues that come up during production.
* [Vket 公式アセット (Official Asset Package)](https://vket.com/docs/official_distribution_package)\
A pack of official Prefabs provided by Vket, including commonly used features such as the Avatar Pedestal, Pickup objects, and video players.\
Using the official Prefabs can save a lot of time, and some Unity functions are available only through them.
