# Tools & Materials

From creating the stinky tofu sea slug itself and preparing the Vket submission to building the VRChat exhibition space,\
I used the following tools and materials.

### Production Tools

* [Blender 5.0](https://www.blender.org/)
* [CLIP STUDIO PAINT (5.0)](https://www.clipstudio.net/tc/)
* [Adobe Photoshop](https://www.adobe.com/tw/products/photoshop.html)

### VRChat / Unity Tools

* [VRChat Creator Companion (VCC)](https://vcc.docs.vrchat.com/) / Unity 2022.3.22f1
* [lilToon (Vket 2.3.2 / Booth 2.3.4)](https://lilxyzw.github.io/lilToon/)
* [Modular Avatar](https://modular-avatar.nadena.dev/) (MA Toggle setup for the Avatar version)
* [AvaPo!｜Unityでポーズ作成＆撮影ツール](https://booth.pm/ja/items/7249970) (Posing & photography tool inside Unity)
* [LuraSwitch2](https://booth.pm/ja/items/1969082) (Setting up interactive gimmicks in the VRChat exhibition space)
